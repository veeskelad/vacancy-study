# Example Job Description — Backend Engineer at NorthStar SaaS

> Synthetic example for demonstrating the skill. Replace with your own JD.

## About the company

NorthStar SaaS is a 60-person B2B startup building a customer onboarding platform used by mid-market SaaS companies. The product helps customer success teams automate the first 30 days of a new account — workflows, in-app guides, usage analytics, and account-health scoring.

The engineering team is 12 people split into three squads (Onboarding Core, Analytics, Integrations). The role is on the Onboarding Core squad — the backend behind the workflow engine and the customer-facing API.

## Responsibilities

- Design and ship features in the backend service (REST + async workers)
- Write clean, tested Python code; participate in code review
- Optimise PostgreSQL queries and design schemas for new features
- Maintain background job processing (Celery / RQ)
- Build and document REST APIs for the frontend and external integrations
- Participate in on-call rotation (one week every six weeks; daytime only)
- Contribute to architecture discussions and tech-debt reduction

## Requirements

- 2+ years of production Python (FastAPI, Flask, or Django)
- Strong PostgreSQL — queries, indexes, migrations
- Experience with REST API design and JSON Schema / OpenAPI
- Comfortable with Docker for local development
- Working knowledge of Redis (caching, queues)
- Git, GitHub Pull Requests workflow
- English at a level sufficient for written async communication

## Nice to have

- Experience with Celery, Sidekiq, or similar task queue
- Kubernetes basics (kubectl, Helm)
- AWS — ECS, RDS, S3, CloudWatch
- Observability tooling — Sentry, Datadog, or open-source equivalents (Prometheus + Grafana)
- Type hints / mypy
- Prior work on multi-tenant SaaS

## Interview format

45-minute conversation: 10 minutes about the team and product, 25 minutes on past experience and a technical discussion, 10 minutes for the candidate's questions.

No coding live; there's a 90-minute take-home assignment after this round if both sides want to proceed.
